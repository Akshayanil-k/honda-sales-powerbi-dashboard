# 🏍️ Honda Bike Sales Analysis Dashboard

## 📌 Overview

This project presents an interactive Power BI dashboard designed to analyze Honda bike sales data.
It provides insights into sales performance, customer preferences, and regional trends to support data-driven decision-making.

---

## 🎯 Project Goals

* Understand sales trends over time
* Identify high-performing regions and markets
* Analyze customer payment behavior
* Evaluate the effect of discounts on revenue and profit

---

## 📊 Key Metrics

* Net Sales Amount
* Gross Profit
* Cost Price
* Order Count
* Customer Ratings

---

## 📁 Dataset Summary

The dataset contains transactional sales data including:

* Order details (ID, Date, Month, Year)
* Location (State, City, Dealer)
* Product details (Bike Model, Color)
* Financial metrics (Sales, Cost, Profit, Discount)
* Payment and additional charges (Insurance, Accessories)

---

## 📈 Dashboard Highlights

* 📊 Sales by State and Payment Mode
* 📉 Monthly Sales Trend Analysis
* 💰 Profit and Cost Breakdown
* 📋 Detailed transaction table
* 🎛️ Interactive filters (Slicers)

---

## 🛠️ Technologies Used

* Power BI (Dashboard & Visualization)
* Microsoft Excel (Data Processing)

---

## 🔍 Insights

* Strong sales performance in major regions like Delhi
* Balanced usage of payment methods (Cash, UPI, Finance)
* Discounts influence purchasing decisions
* Opportunities exist to improve profitability in some regions

---

## 💡 Recommendations

* Focus on high-revenue regions for growth
* Optimize discount strategies
* Increase promotion of finance options
* Improve customer satisfaction and retention
